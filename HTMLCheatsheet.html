    Text Emphasis

    Italics

    The 'em' element adds extra emphasis to the text within the opening and closing tags.
    This shows up as italic text in most browsers.

    Bold

    You can also use the 'strong' element to emphasize text. This shows up as bold text in most browsers.

    Lists

    Unordered Lists

    Unordered lists are marked with bullet points and go in 'ul' tags.
    Each list item goes in an 'li' tag.

    Types of Lists:

    • Unordered lists ('ul)
    • Ordered Lists ('ol')

    Ordered Lists

    Ordered lists are numbered and go in 'ol' tags.
    Each list item goes in 'li' tags.

    Anchors (i.e Links)

    Anchor tags create links to other pages. The URL goes in the 'href' attribute. Go to duckduckgo to search for more information.

    Setting 'target='"_blank" will open the link in a new window. See MDN for more details.

    Anchor tags can also create links to a new location on the same page.
    Instead of putting a URL in the "href" attribute, just use the "id" of the element you want to navigate to. This allows you to jump to the top of the page.

    `div`
